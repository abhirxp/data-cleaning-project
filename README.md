# Data Cleaning Pipeline

A robust Python pipeline for cleaning dirty datasets. This project demonstrates how to handle common data issues such as duplicate entries, missing values, incorrect data types, and inconsistent text formatting using Pandas.

## Features
* **Duplicate Removal:** Identifies and removes duplicate records based on unique identifiers.
* **Missing Value Imputation:** Fills gaps in numeric data using median/mode strategies.
* **Type Conversion:** Fixes string-to-number and string-to-date casting issues.
* **Text Standardization:** Trims whitespace and normalizes text case.
